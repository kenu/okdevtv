# Spring Boot 2

## Requirement
* Java 8+
* Tomcat 8+
* Thymeleaf 3
* Hibernate 5.2

## Spring Framework 5
* Reactive Spring
* Functional APIs
* Kotlin Support

## argument
* Spring Boot 2 `-Dspring-boot.run.arguments`
```
mvn spring-boot:run -Dspring-boot.run.arguments=--server.port=9003
```
