# Spring Boot 2

## Requirement
* Java 8+
* Tomcat 8+
* Thymeleaf 3
* Hibernate 5.2

## Spring Framework 5
* Reactive Spring
* Functional APIs
* Kotlin Support

## IntelliJ Devtools Reload
* Compiler option
  * check `Build project automatically`
* Registry option
  * check `compiler.automake.allow.when.app.running`
