# ES7
- ECMAScript2016

## JavaScript Array.prototype.includes
```js
["Banana", "Orange", "Apple", "Mango"].includes("Mango"); // is true
```

## Exponentiation Operator
```js
let z = 5 ** 2; // 25
let y = Math.pow(5, 2); // 25
```

## Exponentiation Assignment
```js
let z = 5;
z **= 2; // 25
```
