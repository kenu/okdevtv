# PullRequest
* 코드를 주고받는 방법
* GitHub에서 시작

## 절차A
* master 에서 develop 브랜치 생성
* develop 브랜치 수정
* new PullRequest 생성
* base:`master` ← compare:`develop` 생성

## 절차B
* 프로젝트 fork(복사하기)
* 복사한 프로젝트 develop 브랜치 수정
* new PullRequest 생성
* base repository: `원본프로젝트` base: `master` ← head repository: `fork프로젝트` compare:`develop` 생성
