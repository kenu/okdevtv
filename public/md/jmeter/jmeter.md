# jmeter
- http://jmeter.apache.org
- 성능 측정 도구
- java required

## Download and Install
- http://jmeter.apache.org/download_jmeter.cgi

## run with GUI
```
bin/jmeter
```

## run without GUI
```
bin/jmeter -n -t test.jmx -e -l test.jtl -o /tmp/report
```
