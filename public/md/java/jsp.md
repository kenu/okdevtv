# JSP JavaServer Pages
- JSP(Java Server Pages)는 HTML 내부에 Java 코드를 넣어 동적인 웹 페이지를 생성하는 웹 어플리케이션 도구

## Prerequsite
- Java
  - Servlet
- HTTP
  - HTML
  - CSS
  - JavaScript

## WebApp
- Structure
  - Deploy
  - Development
- Lifecycle
## Syntax
- 스크립틀릿 Scriptlet
  - Control Flow
  - Conditional
  - Loop
  - Operator
  - Literal
- 선언자 Declarative
  - <%@ page %>
  - <%@ include %>
  - <%@ taglib %>
- 액션 Action
  - <jsp:include>
  - <jsp:useBean>
  - <jsp:setProperty>
  - <jsp:getProperty>
  - <jsp:forward>
  - <jsp:plugin>
  - <jsp:text>
- 내장 객체 Implicit Object
  - request
    - HttpServletRequest
  - response
    - HttpServletResponse
  - out
    - PrintWriter
  - sessioin
    - HttpSession
  - application
    - ServletContext
  - config
    - ServletConfig
  - pageContext
    - JspWriters
  - page
    - this
  - exception
    - Exception
- 주석 Comment
- 지시자 Directive
- 커스텀 태그 Custom Tag
- EL Expression Language
