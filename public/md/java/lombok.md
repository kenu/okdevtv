# Lombok
- https://projectlombok.org
- Annotation based utils
- @Getter, @Setter, @ToString

## pom.xml
```
   <dependency>
      <groupId>org.projectlombok</groupId>
      <artifactId>lombok</artifactId>
      <version>1.18.32</version>
      <scope>provided</scope>
   </dependency>
```

## Features
@Getter
@Setter
@ToString
@Cleanup
@NotNull

## IDE install
- IntelliJ - Plugins
- Eclipse(STS) - run lombok.jar
