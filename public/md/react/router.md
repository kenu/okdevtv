# React Router
- `react-router-dom`

```html
<BrowserRouter>
  <Routes>
    <Route
      path="/"
      element={
        <div>
          <h1>Hello World</h1>
        </div>
      }
    ></Route>
    <Route
      path="/about"
      element={
        <div>
          <h1>About React</h1>
        </div>
      }
    ></Route>
  </Routes>
</BrowserRouter>
```
