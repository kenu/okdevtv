# Vite
* https://vitejs.dev/
* /vit/, french word for 'fast'

## First vite project
```
# npm 6.x
npm init vite@latest my-vue-app --template vue

# npm 7+, extra double-dash is needed:
npm init vite@latest my-vue-app -- --template vue
```

* template preset list
  * vanilla
  * vanilla-ts
  * vue
  * vue-ts
  * react
  * react-ts
  * preact
  * preact-ts
  * lit-element
  * lit-element-ts
  * svelte
  * svelte-ts
