# nvm
* Node.js Version Manager
* https://github.com/nvm-sh/nvm

## Install
* https://github.com/nvm-sh/nvm#installing-and-updating

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

## install node

```sh
nvm install 20
```

## set default version

```sh
nvm alias default 20
```

## list lts versions

```sh
nvm ls-remote --lts
```
