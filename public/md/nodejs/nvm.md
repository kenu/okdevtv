# nvm
* Node.js Version Manager
* https://github.com/nvm-sh/nvm

## Install
* https://github.com/nvm-sh/nvm#installing-and-updating

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
```

## install node

```
nvm install v20
```

## set default version

```
nvm alias default v20
```

## list lts versions

```
nvm ls-remote --lts
```
