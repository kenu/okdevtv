# pnpm
- Fast, disk space efficient package manager
- https://pnpm.io/

## Install
- `curl -fsSL https://get.pnpm.io/install.sh | sh -`
- `brew install pnpm`
- `npm install -g pnpm`

## Usage

| npm command | pnpm equivalent |
|---|---|
|`npm install` |`pnpm install` |
|`npm i <pkg>` |`pnpm add <pkg>` |
|`npm run <cmd>`|`pnpm <cmd>` |
